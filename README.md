# 💡 Sentiment Analyzer Web App

A full-stack sentiment analysis web application that analyzes user input and classifies it as Positive, Negative, or Neutral with percentage scoring.

## 🚀 Features
- Real-time sentiment analysis
- Percentage-based prediction
- Interactive UI
- Sentiment trend chart (Chart.js)
- History tracking using localStorage
- Dark mode support

## 🛠️ Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- JSON
- Chart.js

## ▶️ How to Run
1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## 📌 Project Purpose
This project was built as a beginner-friendly machine learning web application to demonstrate text analysis, frontend-backend integration, and data visualization.
